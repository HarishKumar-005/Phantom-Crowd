# Contributing to Phantom Crowd

Thank you for your interest in contributing to Phantom Crowd.

We welcome developers interested in:

- Augmented Reality
- Spatial Computing
- Civic Technology
- Smart City Platforms

---

## Ways to Contribute

You can contribute by:

- fixing bugs
- improving UI/UX
- optimizing AR rendering
- improving spatial queries
- enhancing moderation models
- writing documentation

---

## Development Setup

**Clone the repository**

```bash
git clone https://github.com/HarishKumar-005/phantom-crowd
```

**Open project in Android Studio**

```
File → Open → phantom-crowd
```

**Add Firebase configuration**

```
Place google-services.json in the app/ directory
Enable Firestore in your Firebase Console
```

**Run on ARCore supported device**

---

## Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/your-feature`)
3. Commit your changes (`git commit -m 'feat: add your feature'`)
4. Push to the branch (`git push origin feature/your-feature`)
5. Open a pull request

Please ensure code follows Kotlin best practices and passes `./gradlew assembleDebug` before submitting.
